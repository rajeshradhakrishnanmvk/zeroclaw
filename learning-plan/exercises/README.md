# Learning Exercises

This directory contains hands-on exercises for each day of the learning plan.

## Structure

```
exercises/
├── day-1/          # Day 1 exercises
├── day-2/          # Day 2 exercises
├── day-3/          # Day 3 exercises
├── day-4/          # Day 4 exercises
├── day-5/          # Day 5 exercises
├── day-6/          # Day 6 exercises
└── day-7/          # Day 7 exercises
```

## Exercise Guidelines

### Completing Exercises

1. **Read the instructions** in each day's guide carefully
2. **Create the required files** in the appropriate day folder
3. **Document your work** thoroughly
4. **Test your understanding** by explaining concepts in your own words
5. **Review and iterate** if something isn't clear

### Exercise Types

#### 1. Analysis Exercises
- Study existing code
- Document patterns and flows
- Create diagrams and charts
- Compare implementations

#### 2. Configuration Exercises
- Write configuration files
- Test different settings
- Document configuration options
- Create example configs

#### 3. Code Reading Exercises
- Trace code execution
- Understand data flow
- Document architecture
- Identify patterns

#### 4. Hands-On Projects
- Implement custom components
- Build integrations
- Test functionality
- Document your work

### Tips for Success

1. **Take your time** - Don't rush through exercises
2. **Ask questions** - Write down what you don't understand
3. **Experiment** - Try variations and see what happens
4. **Document** - Write clear notes for future reference
5. **Review** - Go back and review previous exercises

### Exercise Checklist Template

For each exercise, track:

- [ ] Exercise started
- [ ] Instructions read and understood
- [ ] Required files created
- [ ] Exercise completed
- [ ] Documentation written
- [ ] Self-review done
- [ ] Questions noted for follow-up

## Getting Help

If you get stuck:

1. Review the relevant day's guide
2. Check the documentation in `docs/book/src/`
3. Look at existing implementations in the codebase
4. Review test files for examples
5. Take a break and come back with fresh eyes
