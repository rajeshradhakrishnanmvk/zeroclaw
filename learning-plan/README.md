# ZeroClaw 7-Day Learning Plan

Welcome to your structured learning journey through the ZeroClaw autonomous agent runtime!

## Overview

ZeroClaw is a Rust-first autonomous agent runtime optimized for performance, efficiency, stability, extensibility, sustainability, and security. This 7-day plan will guide you through understanding the architecture, core concepts, and hands-on experimentation.

## Prerequisites

- Basic Rust knowledge (ownership, traits, async/await)
- Familiarity with CLI tools
- Understanding of AI/LLM concepts (helpful but not required)

## Learning Structure

Each day has:
- **Theory**: Core concepts and architecture
- **Exploration**: Code reading and analysis
- **Practice**: Hands-on exercises
- **Resources**: Links to relevant documentation

## Daily Schedule

- **Day 1**: Project Setup & Architecture Overview
- **Day 2**: Configuration System & Traits
- **Day 3**: Providers & Memory Systems
- **Day 4**: Channels & Communication
- **Day 5**: Tools & Runtime
- **Day 6**: Security & Observability
- **Day 7**: Integration & Extension

## Getting Started

1. Read through each day's guide in order
2. Complete the exercises in the `exercises/` folder
3. Take notes in the `notes/` folder
4. Refer to `reference/` for quick lookups

## Progress Tracking

Use the checklist in each day's guide to track your progress.
